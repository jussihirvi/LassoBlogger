First Things First

Welcome to LassoBlogger v0.5.0 - the first public version. I find it hard to believe this project has come this far.

I started this, because I need a blogware, and don't know php well enough to feel comfortable with using php software. Many things in this blogware are copied or adapted from the justly famous WordPress blogware (written in php). I want to thank them very much and hope they don't mind. Probably not - I guess that's what open source is about.

Installation

  1. Unpack the package in an empty directory
  2. Open up config-sample.lasso with a text editor (on Windows, choose WordPad or similar) and fill in your database connection details
  3. Save the file as config.lasso
  4. Upload everything to your server.
  5. Launch /admin/install.lasso in your browser. This should setup the tables needed for your blog. If there is an error, double check your config.lasso file, and try again. If it fails again, please read the dev blog (though now it's still empty) or contact the developer - I hope you don't just give up.
  6. Note the password given to you.
  7. The install script should then send you to the admin pages for login. Sign in with the username admin and the password generated during the installation. You can then click on 'Users', etc., to change the password.

System Recommendations

  • Lasso version 6 or higher (may work on v5, but is not tested; I have developed it on L6.0.6, and "strict syntax" was off)
  • MySQL, may even work on other SQL db's, but is not tested

User Levels

The user level scheme may be a bit coarse at this stage. The admin page "Edit users" helps you grade the levels. The theoretical levels range from 0 to 10, but essentially the relevant levels are:

  • 0 - Can edit his/her own user info
  • 2 - Can post, edit, and delete his/her own posts
  • 5 - May edit links and categories; sees list of users
  • 8 - Admin - may edit options, and edit posts, comments and user info of all users
  • 10 - Max value

In the usual case you may want a team of level 2 users (writers) except for you. You may also want one or two assistant admins (level 8 would be good for that).

There is no registration page as yet for visitors.

Final notes

This blogware is considered beta, so don't be surprised, if you stumble on bugs or stupid things. When that happens, I really hope you pass the word - at the moment this means emailing me (manager of the project at LassoForge), or writing a comment on the "official" LassoBlogger dev blog.

Future

I will probably continue doing things to this sw, as I start to use it for my own purposes. Most of the options, for example, are now inoperative, though you can see them and edit them on the admin pages (the status of each option - in use or not - is also stated there).

I hope you would do the same. When you find out you miss some feature, why not build a patch, and send it to me, so it could be shared to all the other users. ... Or at least tell me what seems to be missing.

There are lots of things that would need doing. It would be great if you would collaborate. The documentation is very bad, but I will try to provide as much information as possible, in the first place using the LassoBlogger dev blog.

About the name

LassoBlogger doesn't sound very imaginative, does it? I could have thought out something more poetical, like Bloghead, or Blogjam, but I really don't think this blogware is very imaginative at the moment. It only tries to be a decent blog solution and not stumble too much. Maybe it will become more personal in the future, but I feel that really depends on you more than me. The name may change then, too, like those Indian names.
